プライバシーポリシー

Hiyolingo（ひよリンゴ）英単語帳 ／ 最終更新: 2026-07-05

Hiyolingo は、家族内での英語学習を目的とした個人運営の英単語帳ツールです。営利目的ではなく、 第三者への情報提供・広告配信は行いません。本ポリシーは、本ツールおよび連携するカスタムGPTが どのような情報を扱うかを説明します。

1. 取り扱う情報

  • 利用者が登録する英単語とその関連情報（意味・品詞・例文・関連語・イディオム・レベル・タグなど）。
  • クイズの進捗など、閲覧端末のブラウザ内（localStorage）にのみ保存される情報。サーバーには送信されません。

氏名・メールアドレス・連絡先などの個人を特定する情報は収集しません。アクセス解析・トラッキング・Cookie による追跡は行いません。

2. 情報の保存先と公開範囲

  • 登録した単語データは Notion のデータベースに保存されます（データの正本）。
  • そのデータは公開リポジトリ github.com/kajisaden/hiyolingo に同期され、 GitHub Pages 上の本サイトで配信されます。
  • ご注意：公開リポジトリ／公開サイトに載るため、登録した単語や例文は誰でも閲覧できる状態になります。 個人情報・秘密の内容は登録しないでください。

3. カスタムGPT（ChatGPT）との連携

  • 連携するカスタムGPTは、利用者が入力した英単語をもとに意味・例文などを生成し、 Notion API 経由で上記データベースに1件登録します。
  • この登録に必要な範囲（対象データベースへの読み取り・追加）以外の権限は付与していません。ユーザー情報の読み取り権限も付与していません。
  • ChatGPT 側での入力・生成の扱いは、OpenAI の利用規約・プライバシーポリシーに従います。

4. 第三者提供

本ツールは、上記（Notion・GitHub・ChatGPT）以外の第三者へ情報を提供しません。広告・解析目的の外部送信も行いません。

5. データの削除

登録した単語は、Notion データベース上で該当行を削除することでいつでも消去できます。削除後は次回の同期でサイトからも取り除かれます。

6. 改定

本ポリシーは必要に応じて改定することがあります。重要な変更がある場合は本ページを更新します。

7. お問い合わせ

本ツールおよび本ポリシーに関するお問い合わせは、リポジトリ運営者（GitHub: @kajisaden）までご連絡ください。
